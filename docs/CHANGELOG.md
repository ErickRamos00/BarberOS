# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

## [1.0.0] - 2026-04-04

### ✨ Lançado
- Backend completo com Express.js
- API REST com 7+ rações principais
- Autenticação com JWT
- Banco de dados SQLite
- Dashboard do dono
- Sistema de agendamento online para clientes
- Gerenciamento de barbeiros, serviços e clientes
- Relatórios financeiros
- Identidade visual customizável
- Frontend responsivo com vanilla JS
- Rate limiting
- Validações completas
- Tratamento de erros global
- Scripts de seed e teste
- Documentação completa
- Pronto para produção

### 🔐 Segurança
- JWT com expiração de 30 dias
- Bcrypt para hashing de senhas
- CORS configurável
- Rate limiting por IP
- Validação de entrada em todos os endpoints
- Sanitização de dados
