## 🚀 Como Usar

### Desenvolvimento Local

```bash
# Instalar
npm install

# Rodar (modo dev)
npm run dev

# Rodar (modo produção)
npm run production

# Testar
npm run test

# Seed com dados demo
npm run seed
```

### Publicar

Escolha uma das **4 opções profissionais:**

1. **DigitalOcean App Platform** (Recomendado)
   - Mais fácil, gerenciado, bom preço
   - Veja DEPLOYMENT.md - Seção "Opção 1"

2. **Heroku**
   - Gratuito até X uso
   - Git push → Deploy automático
   - Veja DEPLOYMENT.md - Seção "Opção 2"

3. **VPS Linux** (DigitalOcean, Linode, AWS)
   - Controle total, mais barato
   - PM2 + Nginx + Certbot
   - Veja DEPLOYMENT.md - Seção "Opção 3"

4. **Docker** (Qualquer plataforma)
   - Containerizado, escalável
   - `docker-compose up`
   - Veja DEPLOYMENT.md - Seção "Opção 4"
